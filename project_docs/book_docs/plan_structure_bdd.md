## Plan de la structure de la base de données

Ce fichier contient la structure de la base de données textuelle avec les actes, les numéros de chapitres, les noms des chapitres ainsi que ceux vers lesquels ils renvoient.

Le classement est effectué par ordre d'apparition des chapitres.

#### Légende :

**Titre du chapitre**  
-> Chapitre vers lequel il renvoie

#### Informations de jeu :

Certains chapitres permettent au joueur d'obtenir un fragment.  
Ces chapitres sont indiqués par la mention **_(-> obtention fragment)_** dans leur titre.  
Le fait d'avoir récupéré un fragment modifie les textes des certains chapitres de l'histoire.
